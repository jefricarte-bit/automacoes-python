# automacoes-python
# Data Extraction & Automated Reporting

This Python script automates the process of data retrieval, spreadsheet organization, and delivering management reports via email.

## Tech Stack
* Python 3
* Pandas (Data manipulation)
* Openpyxl (Excel integration)

## Problem Solved
Eliminates hours of manual work by transforming raw data into a production-ready report, sent directly to decision-makers.
